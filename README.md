## Core Features Overview

- **Luxury Bilingual Experience**  
  Seamlessly switch between English and Indonesian across the entire application, providing a refined and accessible experience for global travelers.

- **Custom Reservation Inquiry Dashboard**  
  A dedicated VIP Concierge reservation system featuring dynamically generated receipt numbers, booking management, inquiry tracking, and personalized travel requests.

- **Destination Archive & Detail Exploration**  
  Browse a curated collection of premium travel destinations with immersive detail modals, destination highlights, luxury amenities, and travel insights.

- **Micro-Interaction & Animation System**  
  Elegant transitions, smooth page interactions, hover effects, and premium motion design elements that enhance user engagement and visual storytelling.

## Modular Project Structure

A well-organized and scalable `/src` architecture designed for maintainability and future expansion.

```text
src/
├── components/       # Reusable UI components
├── pages/            # Application pages and route views
├── data/             # Destination datasets and localization content
├── hooks/            # Custom React hooks
├── utils/            # Utility and helper functions
├── assets/           # Images, icons, and static resources
├── styles/           # Global styling and theme configuration
├── contexts/         # Global state and language management
├── App.tsx           # Main application component
└── main.tsx          # Application entry point
